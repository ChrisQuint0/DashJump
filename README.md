# DashJump

DashJump is a high-octane, pixel-art arcade game where players must master the art of timing and precision. Navigate through a series of challenging obstacles using dash and jump mechanics in this fast-paced, addictive mini-game.

![DashJump Preview](preview.png)

## Play the game

https://dash-jump.vercel.app

## Features

- **Dynamic Gameplay**: Smooth dash and jump mechanics for precise control.
- **Pixel Art Aesthetic**: Retro-inspired visuals with a vibrant color palette.
- **Challenging Obstacles**: Increasingly difficult hurdles to test your reflexes.
- **Responsive Design**: Playable on both desktop and mobile devices.
- **Instant Play**: Web-based deployment for quick access anywhere.

## Tech Stack

- **[Phaser 3](https://phaser.io/)**: A fast, free, and fun open-source framework for Canvas and WebGL HTML5 games.
- **[Vite](https://vitejs.dev/)**: Next-generation frontend tooling for a fast development experience.
- **JavaScript (ES6+)**: Modern JavaScript for game logic and state management.
- **CSS3**: Vanilla CSS for layout and styling.
